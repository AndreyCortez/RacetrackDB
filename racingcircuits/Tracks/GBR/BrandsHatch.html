Open Questions
Rallycross - I'm certain usage of the RX offroad parts ended with 2005-06 since the access roads from the paved track vanish in recordings. The question remains if 1981 was the first RX event held at the track, sources indicate them happening as early as 1976.
Kart - I'm uncertain whent the small kart track was build. I was able to narrow it down to between 1970 and 90.
Clearways/Southbank cut - Sometime after the GP extension a link was build allowing for a track variant that would run only the GP loop. Going by track maps from the period it seems to have been build around 66. I've found nothing on it apart pictoral evidence.
Directionality - When was the last time Brands was run Counterclockwise? I found pictures of bikes going CCW as late as 69.
