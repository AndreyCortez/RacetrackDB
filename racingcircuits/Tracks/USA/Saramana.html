Further Details
The timeline was build using a set of aerial image covering the time from 1950 to 1970. The changes under each year should be viewed as snapshots, rather than the true date the track was changed. Judging from other sources the page still covers all layouts of the track.
I found little information on the ½ mile oval. Aerial images show it being build around the same time as the ¼ mile. I don't know when it stopped operating, but photos from 1957 show a path running through the already overgrown turns.
