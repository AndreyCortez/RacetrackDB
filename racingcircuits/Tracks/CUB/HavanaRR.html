The 1905 GP was run in two legs. The outbound leg to San Crist贸bal appears to have been run via Guanajay, while the inbound leg went via Ceiba del Agua. The distances of the two routes confirm to the distances given by Tony Kaye
The 1906 GP was run on the same course over 4 laps. The lap distance is about 8km longer than the Guanajay leg. The start has been moved closer to Havana so the distances agree.
For 1905 Tony Kaye mentions Sprint races held two days after the main GP, while Darran Galphin mentions record attempts being held two days prior. Both events seemingly taking place over the same stretch of road. I'm unsure if these two mentioned events arn't the same given the symmetric spacing around the main race and same location.
